## 开发环境

- Win10
- 编译环境 Ubuntu 18.04.4 LTS（Windows WSL）
- 运行环境 Bochs 2.6.11

## 注意

强烈建议在Linux环境下开发，

至少编译环节要在Linux环境下，如果使用Windows编译将会带来大量意想不到的坑
